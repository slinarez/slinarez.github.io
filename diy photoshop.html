Beach Scene
Using P5 coding techniques I created multiple brushes using simple shapes to allow the user to create a beach scene.
1 = yellow circle , 2 = white brush/eraser, 3 = light blue brush, 4 = dark blue brush, 5 = light blue square, 6 = gray brush, 7 = biege small square, 8 = opaque orange small squares, 9 = opaque orange big squares, 0 = purple squares
G = sand png, X = clears screen, P = saves screen
