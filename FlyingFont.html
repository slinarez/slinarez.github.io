For this exercise I implemented code that allows users to interact with it. You can click and hold keys to change what is happening on screen. I used key events and properties, added orginal font and text as well.
Pressing any key while clicking will show make a different phrase appear on screen Pressing B will toggle the circle background-color Pressing P will allow you to click and see a purple background appear and pressing X will cancel the purple background.
